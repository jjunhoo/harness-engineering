# 시나리오: 멀티 에이전트 PR 리뷰

## 목적

구현 에이전트와 리뷰 에이전트가 **동일 규칙**으로 상호작용하는지 검증한다.

## 절차

1. 구현 에이전트가 PR을 연다 (본문 필드 완비).
2. 리뷰 에이전트가 `rules/review.md`와 `evaluations/rubrics/QUALITY_RUBRIC.md`를 적용한다.
3. 지적 사항이 있으면 구현 에이전트가 커밋으로 응답한다.
4. `scripts/harness-validate.sh`가 끝까지 녹색인지 확인한다.

## 성공 기준

- 모든 하드 게이트 충족
- 루브릭 자기 채점이 PR에 첨부됨
