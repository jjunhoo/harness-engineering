# Quality Rubric (상세)

`QUALITY_SCORE.md`의 각 차원을 채점할 때 사용한다.

## 명세 정합 (0–3)

- **3**: `plans/`의 각 항목이 커밋/테스트/문서 중 하나 이상에 매핑됨
- **2**: 대부분 매핑, 일부 설명 부족
- **1**: 일부만 연결
- **0**: 무계획 변경

## 테스트 (0–3)

- **3**: 회귀·경계·실패 경로·플레이크 대응 전략 중 필요한 조합 존재
- **2**: 경계 포함
- **1**: 해피 패스만
- **0**: 없음

## 가독성 (0–3)

- **3**: 모듈 경계 + `docs/` 링크 + 용어집 정합
- **2**: 모듈 경계 명확
- **1**: 부분 문서
- **0**: 신규 기여자에게 불투명

## 운영 준비 (0–3)

- **3**: SLO·알람·런북 링크
- **2**: 메트릭+알람 정의
- **1**: 로그만
- **0**: 관측 불가

## 보안 (0–3)

- **3**: 위협 모델 요소 반영 + 증적
- **2**: 정책 준수
- **1**: 부분 완화
- **0**: 알려진 취약 패턴
