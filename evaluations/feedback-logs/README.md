# Feedback Logs (에이전트 읽기용 샘플)

`docs/LOG_FEEDBACK.md`에 따라, **마스킹·샘플링된** 로그 조각을 둘 수 있는 위치다.

## 규칙

- 원본 운영 로그 전체를 커밋하지 않는다.
- 토큰·이메일·전화·주소 등 PII는 제거 또는 해시한다.
- 파일명에 환경·날짜·이슈 ID를 포함한다(예: `staging-2026-04-20-auth-spike.jsonl.sample`).

## 용도

- 멀티 리뷰·트리아지 재현
- `LOG_BASED_TRIAGE` 시나리오 점검
- 회귀 테스트의 입력 골든 파일(비식별화된 경우)
