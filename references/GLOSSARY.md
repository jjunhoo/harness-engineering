# 용어집

| 용어 | 정의 |
|------|------|
| Harness | AI가 소프트웨어를 생산·개선하기 위한 규칙·자동화·지식의 총체 |
| Plan | `plans/`의 실행 단위 문서 |
| Skill | `skills/`의 반복 가능한 절차 정의 |
| Rule | `rules/` 및 CI로 강제되는 제약 |
| Evaluation | 테스트·루브릭·시나리오 등 품질 증명 |
| Agent legibility | 에이전트가 탐색·추론 비용을 낮추는 구조 |
| Hard gate | 점수와 무관하게 머지를 막는 조건 |
